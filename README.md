# NotePet

桌面便條搭子 — 一個住在螢幕上的小動物，幫你管理便條和提醒。

### 右鍵選單

- **撫摸** — 和搭子互動（有冷卻時間，冷卻中會用對話框提醒你）
- **玩耍** — 和搭子玩耍
- **隱藏到系統匣** — 把搭子收到工作列右下角，點擊系統匣圖示可叫回

### 心情系統

搭子依使用頻率有三種心情，會影響外觀上的心情圖示：

| 心情 | 圖示 | 說明 |
|---|---|---|
| 開心 | ❤️ | 最近有互動 |
| 普通 | 😐 | 一般狀態 |
| 難過 | 😭 | 太久沒互動，搭子會透過對話框抱怨 |

### 對話框

搭子左上方的像素風對話框會顯示：
- 撫摸 / 玩耍冷卻中的提示
- 難過時的抱怨訊息
- 喝水提醒

多則訊息會自動排隊依序顯示，每則 3.5 秒後自動關閉。

## 便條面板

點擊搭子即可開啟，再次點擊或按 **Esc** 關閉。

### 便條操作

| 操作 | 動作 |
|---|---|
| **點擊便條卡片** | 在面板右側開啟編輯器 |
| **雙擊便條卡片** | 在浮動視窗開啟（可移動） |
| **右鍵便條卡片** | 開啟選單（刪除、複製等） |
| **拖曳便條卡片** | 重新排序 |

## 便條編輯器

### 基本欄位

- **標題** — 必填。支援快速輸入語法（見下方說明）
- **內容** — 多行文字
- **標籤** — 可多選，或從標籤管理新增

### 提醒功能

勾選 **提醒** 開關後設定日期時間，時間到時便條會自動彈出浮動視窗。
<img width="275" height="314" alt="image" src="https://github.com/user-attachments/assets/599df4b6-15c1-4ac3-8001-e851af927f82" />

重複提醒間隔選項：不重複 / 每天 / 每週 / 每月。

### 標題快速輸入語法

在標題欄的**最開頭**輸入 `-`，後面接日期（可省略）和時間，即可快速設定提醒。  
儲存後自動解析出標題、設定提醒時間，並加上「提醒」標籤。

#### 格式總覽

```
-[日期] [時間] 標題文字

```

各元素**可任意排列順序**，唯一規則是整行必須以 `-` 開頭。
<img width="276" height="313" alt="image" src="https://github.com/user-attachments/assets/6f7d6d17-5cb7-4a25-826b-c6956a7aa619" />

---

#### 前綴

| 格式 | 說明 |
|---|---|
| `-` | 整行第一個字元必須是 `-`，後接一個空格或直接接元素 |

---

#### 時間（必填）

時間分隔符號可用 `:` 或 `.`，am/pm 大小寫不限。

| 格式 | 範例 | 說明 |
|---|---|---|
| `H:MM` | `9:00`、`14:30` | 24 小時制，冒號分隔 |
| `H.MM` | `9.00`、`14.30` | 24 小時制，句點分隔 |
| `H:MMam` / `H:MMpm` | `9:00am`、`2:30pm` | 12 小時制，am/pm 緊接時間或空一格皆可 |
| `H.MMam` / `H.MMpm` | `9.00am`、`2.30PM` | 12 小時制，句點版本 |

> **注意：** `12am` = 午夜 00:00；`12pm` = 中午 12:00

---

#### 日期（可省略，預設為今天）

| 格式 | 範例 | 說明 |
|---|---|---|
| `今天` | `今天` | 當天日期 |
| `明天` | `明天` | 隔天日期 |
| `月/日` | `6/15`、`12/1` | 斜線分隔，月份和日期可為 1–2 位數 |
| `月/日(備註)` | `6/15(六)`、`12/1(週一)` | 括號內可填任意備註（如星期），會自動忽略 |
| `MMDD` | `0615`、`1201` | 四位數緊湊格式，月份需為 01–12、日期 01–31 |

> 年份一律使用當年，不支援跨年指定。

---

#### 完整範例

| 輸入 | 解析結果 |
|---|---|
| `-14:00 開會` | 今天 14:00，標題「開會」 |
| `-9:00am 晨會` | 今天 09:00，標題「晨會」 |
| `-明天 9:00 早晨例會` | 明天 09:00，標題「早晨例會」 |
| `-今天 11.30am 午餐約會` | 今天 11:30，標題「午餐約會」 |
| `-6/15 14:00 會議記錄` | 6 月 15 日 14:00，標題「會議記錄」 |
| `-6/15(六) 14:00 週末會議` | 6 月 15 日 14:00，標題「週末會議」 |
| `-0615 1400 備忘錄` | 6 月 15 日 14:00，標題「備忘錄」 |
| `-2.30pm 6/20 牙醫` | 6 月 20 日 14:30，標題「牙醫」（順序任意）|

## 浮動便條視窗

提醒觸發時，便條會以浮動視窗出現在桌面上：

- 可拖曳移動到任意位置
- 獨立顯示，不會被其他視窗遮住
- 修改後儲存，同步回主列表

## 設定

點擊面板右上角 **⚙** 開啟設定。

### 外觀

| 設定 | 說明 |
|---|---|
| 字體大小 | 11–20px，拖動 slider 即時預覽 |
| 字型 | 微軟正黑體 / 新細明體 / 標楷體 / 微軟雅黑 / Arial |
| 色系風格 | 預設（奶黃）/ 深夜（深藍）/ 薄荷（淺綠）/ 玫瑰（粉紅） |
| 時間格式 | 24 小時制 / 12 小時制 |
| 面板尺寸 | 自訂寬高（寬 200–800、高 160–800） |

### 搭子

| 設定 | 說明 |
|---|---|
| 搭子透明度 | 10%–100%，即時預覽 |
| 喝水提醒 | 開啟後定時用對話框提醒你喝水 |
<img width="240" height="188" alt="image" src="https://github.com/user-attachments/assets/e4ebe510-eebd-46a8-8689-2a2b31ef5d13" />

| 提醒間隔 | 1–1440 分鐘（預設 60 分鐘） |
| 更換搭子圖片 | 開啟搭子圖片管理，可上傳自訂圖片 |

### 系統

| 設定 | 說明 |
|---|---|
| 開機自動啟動 | 登入 Windows 時自動啟動 NotePet |
| 管理標籤 | 新增 / 刪除標籤 |
| 關閉 NotePet | 完全結束程式 |

### 按鈕

- **取消** — 捨棄未儲存的變更，還原即時預覽
- **套用** — 儲存所有設定並套用

---

## 標籤系統

- 面板頂端顯示所有標籤，點擊可篩選便條
- 點「全部」回到全部顯示
- 透過 **設定 → 系統 → 管理標籤** 新增或刪除標籤

---

## 搭子圖片管理

**設定 → 搭子 → 更換搭子圖片**

- 最多支援 3 個搭子（預設 + 自訂 1 + 自訂 2）
- 可分別設定「一般圖片」和「被抓圖片」（拖曳時顯示）
- 點擊搭子名稱旁的單選按鈕切換當前使用的搭子

---

## 系統匣

搭子隱藏後會出現在 Windows 工作列右下角的系統匣，雙擊圖示可讓搭子重新出現。
